# 🎨 Art Gallery Marketplace

A full-stack **Art Gallery E-Commerce Website** built using **Python Flask, SQLite, Bootstrap, HTML, CSS, and JavaScript**.

This project allows users to browse beautiful artworks, register/login, view product details, add items to cart, and manage their account.

---

# 🚀 Features

## 👤 User Features

- User Registration
- User Login / Logout
- My Account Page
- Browse Products
- View Single Product Details
- Add to Cart
- Remove from Cart
- Responsive UI

---

## 🎨 Product Features

- Homepage Featured Artworks
- Products Listing Page
- Product Detail Page
- Dynamic Image Loading from Database
- Categories Section

---

## 🛒 Cart Features

- Add Items to Cart
- Remove Items from Cart
- Auto Total Price Calculation

---
